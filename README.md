# KOVO

KOVO ist eine immersive Landingpage für Premium‑Bergschuhe der Alpine Division. Die Seite inszeniert das Produkt mit
cineastischem Hero‑Video, technisch anmutender UI und klarer Storyline von Performance, Engineering und Handwerk.

## ✨ Highlights der Web‑Experience

- Video‑Hero mit Markeninszenierung und HUD‑Overlay
- Interaktiver Produkt‑Viewer mit Ebenen (Exterior/Core/Traction)
- Mission‑Reports und Field‑Dispatches als Social‑Proof‑Sektion
- Nachhaltigkeits‑Bereich mit Kennzahlen und Video‑Frame
- FAQ‑Akkordeon, Pre‑Order‑CTA und Newsletter‑Formular

## 🧰 Tech‑Stack

- Vite
- HTML/CSS/JavaScript

## 🚀 Lokale Entwicklung

```bash
npm install
npm run dev
```
